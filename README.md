ncurses4j
=========

ncurses4j is a Java binding to the ncurses library.


Installation
------------

Amend your `build.sbt`:

    resolvers += "valotrading" at "http://valotrading.github.com/maven"

    libraryDependencies += "ncurses4j" % "ncurses4j" % "0.1.0"
